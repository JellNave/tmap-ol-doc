# 简介

# TMap是什么

TMap.js是基于OpenLayers (opens new window)最新版本（7.1.0）封装的常用GIS功能的用JavaScript编写的工具类方法集合

# 为什么选用OL作为底层方法

OpenLayers作为开源JavaScript GIS库之一具有高性能且功能丰富等特点，WEB端GIS普遍选用OL进行开发

想了解更多OL知识，请移步这里 OL简明教程

# 为什么要编写TMap.js

OpenLayers虽然功能强大，使用灵活，但是对于前端工程师来说需要一定的学习成本，在地图相关需求较为紧急的情况，难以及时的实现相关功能，通过使用TMap.js，前端工程师只需要简单的调用即可实现一个功能

团队内部所使用的的旧版本GIS库存在一系列问题：方法封装不够灵活，实现较为复杂，代码量过于庞大，维护十分困难，所支持OL版本过低，升级迁移困难

所以TMap.js诞生了！

ps：

考虑到项目兼容性，选用JavaScript编写而非TypeScript编写

# 为什么要选择函数式封装而非面向对象

TMap.js其实更像是一系列功能的封装，更偏向于功能的实现，即一个方法实现一个功能或者多个方法组合成一种功能。通过函数式编程及es6export语法，调用者可以更清晰的通过import按需引入相应的方法来实现某一功能，若需要调整到某一功能，维护者也能及时的定位到方法进行维护
