# API / fitInFeaturesOrGeometries

# 类型

(TMap:TMap,featuresOrGeometries:featuresOrGeometries ,option:option )=> void

(地图实例化对象,要素或几何图形数组，相关配置)=>无返回值

# 描述

动态改变地图视图，使地图恰好能显示n个y要素或者n个几何图形,有动画效果

# 参数

# TMap

  • 类型：Object
  • 描述：地图实例对象（初始化地图返回值之一）
  • 是否必须：是
  • 详细属性：Map对象，详细请参照ol/Map

# featuresOrGeometries

  • 类型：Array.<ol/Feature | ol/Geometry>
  • 描述：Feature数组或Geometry数组
  • 是否必须：是
  • 详细属性：数组形式包裹的Feature对象或Geometry对象，但是注意不要混用，一个数组内的对象只能是同一种，若你得到了一批不种类的对象，可以利用Feature上的getGeometry()转换为Geometry对象，然后再作为参数传入

# option

  • 类型：Object
  • 描述：适应方法相关配置
  • 是否必须：否
  • 详细属性：Object包含以下属性
    属性名 是否必须 类型 默认值 描述
    padding 否 Array.<number> [60,60,60,60] 适应图形与容器边界距离，数组形式表示四个方向的padding值，顺序为[top,right,bottom,left]
    size 否 Array.<number> undefined 像素大小适应范围，eg[100,100]
    durationTime 否 number 300（单位:ms） 地图视图改变的动画过渡时间
    maxZoom 否 number undefined 触发缩放的最大缩放值，即上限，表示就算范围足够小，zoom到达上限也不会继续放大
