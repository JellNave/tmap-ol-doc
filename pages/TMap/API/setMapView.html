# API / setMapView

# 类型

(TMap:TMap,option:option )=>void

(地图实例化对象，视图配置项)=>无返回值

# 描述

改变当前视图相关配置中的一项或多项，比如中心位置，缩放等，（无动画效果，强制改变）

# 参数

# TMap

  • 类型：Object
  • 描述：地图实例对象（初始化地图返回值之一）
  • 是否必须：是
  • 详细属性：Map对象，详细请参照ol/Map

# option

  • 类型：Object
  • 描述：视图相关配置项，不传保持原样
  • 是否必须：否
  • 详细属性：Object包含以下属性
    属性名 是否必须 类型 默认值 描述
    projection 否 string 当前View对象projection值 地图使用的投影坐标系，可选EPSG:4326,EPSG:3857
    center 否 Array.<number> 当前View对象center值 地图中心点经纬度（遵循EPSG:4326 coord数据格式）[centerLng,centerLat]
    zoom 否 number 当前View对象zoom值 地图初始化视图缩放值
    maxZoom 否 number 当前View对象maxZoom值 地图最大缩放层级
    minZoom 否 number 当前View对象minZoom值 地图最小缩放层级
    extent 否 Array.<number> 当前View对象extent值 限制地图可见范围，数组[minx, miny, maxx, maxy]中四个值依次为X方向最小值，y方向最小值，x方向最大值，y方向最大值，通俗理解就是经纬度的最值
