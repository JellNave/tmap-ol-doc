# API / getDistance

# 类型

(startLngLat:startLngLat,endLngLat:endLngLat, radius:radius )=>number

(起点坐标,终点坐标,地球半径)=>距离长度

# 描述

计算两个坐标点的m数距离

# 参数

# startLngLat

  • 类型：Object
  • 描述：起始点坐标对象
  • 是否必须：是
  • 详细属性：Object包含以下属性
    属性名 是否必须 类型 默认值 描述
    lng 是 number undefined 坐标点经度，coord数组第一个值，必须为数字
    lat 是 number undefined 坐标点纬度，coord数组第二个值，必须为数字

# endLngLat

  • 类型：Object
  • 描述：终止点坐标对象
  • 是否必须：是
  • 详细属性：Object包含以下属性
    属性名 是否必须 类型 默认值 描述
    lng 是 number undefined 坐标点经度，coord数组第一个值，必须为数字
    lat 是 number undefined 坐标点纬度，coord数组第二个值，必须为数字

# radius

  • 类型：number
  • 描述：地球半径长度
  • 是否必须：否
  • 详细属性：所选用的投影坐标系所基准的地球球体半径，默认为wgs-84即EPSG:4326投影坐标系的球体半径：6371008.8（单位m）

# 返回值

# number

  • 类型：number
  • 描述：两点距离
  • 详细属性：传入两个坐标之间的实际距离，单位为m
