# API / animateMapView

# 类型

(TMap:TMap,option:option )=>void

(地图实例化对象，视图配置项)=>无返回值

# 描述

改变当前视图相关配置中的一项或多项，比如中心位置，缩放等，（有动画效果，可指定时长与动画效果）

# 参数

# TMap

  • 类型：Object
  • 描述：地图实例对象（初始化地图返回值之一）
  • 是否必须：是
  • 详细属性：Map对象，详细请参照ol/Map

# option

  • 类型：Object
  • 描述：视图相关配置项，不传保持原样
  • 是否必须：否
  • 详细属性：Object包含以下属性
    属性名 是否必须 类型 默认值 描述
    center 否 Array.<number> 当前View对象center值 地图中心点经纬度（遵循EPSG:4326 coord数据格式）[centerLng,centerLat]
    zoom 否 number 当前View对象zoom值 地图初始化视图缩放值
    durationTime 否 number 300（单位:ms） 地图视图改变的动画过渡时间
    easing 否 string inAndOut 地图动画效果 [easeIn：由慢到快]，[easeOut：由快到慢]，[easeOut：先慢后快再慢]，[linear：平滑匀速]，[upAndDown：先慢后快再慢，但是会恢复原样]

# 返回值

# void
