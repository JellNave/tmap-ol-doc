# API / getAllLayers

# 类型

(TMap:TMap )=>Array:Array

(地图实例化对象)=>层实例化对象数组

# 描述

获取当前地图上的所有层（Layer对象）

# 参数

# TMap

  • 类型：Object
  • 描述：地图实例对象（初始化地图返回值之一）
  • 是否必须：是
  • 详细属性：Map对象，详细请参照ol/Map

# 返回值

# Array

  • 类型：Array.<Object>
  • 描述：地图层实例化对象的数组，若无额外图层则只有一层底图图层
  • 详细属性： Object为Layer对象，详情请查看ol/Layer
