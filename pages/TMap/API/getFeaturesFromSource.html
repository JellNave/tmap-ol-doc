# API / getFeaturesFromSource

# 类型

(Source:Source )=>Array:Array (要素所在地图层)=>要素实例数组

# 描述

获取当前地图层上的所有要素

# 参数

# Source

  • 类型：Object
  • 描述：数据源实例对象（添加矢量图层返回值之一）
  • 是否必须：是
  • 详细属性：Source对象，详细请参照ol/Source

# 返回值

# Array

  • 类型：Array.<Object>
  • 描述：地图要素实例化对象的数组，若当前数据源无要素则返回[]
  • 详细属性： Object为Feature对象，详情请查看ol/Feature
