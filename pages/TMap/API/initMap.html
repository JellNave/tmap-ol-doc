# API / initMap

# 类型

(option:option)=>Object:Object

(地图初始配置)=>{地图实例化对象，地图基层实例化对象}

# 描述

通过一系列配置初始化渲染一个地图到容器内

# 参数

# option

  • 类型：Object
  • 描述：地图初始化传入的相关初始配置
  • 是否必须：是
  • 详细属性：Object包含以下属性
    属性名 是否必须 类型 默认值 描述
    targetContainerId 是 string undefined 地图初始化容器id
    (注：请勿在该容器内添加类名为ol-viewport的元素节点,具体原因请查看OL简明教程/TMap源码解析)
    baseMap 否 string undefined 地图底图资源配置名称，即mapConfig.js中baseMapConfig[baseMap],根据配置键名而定,已存在的配置：["vector"：矢量地图]， ["image":影像地图]， ["dark":夜景地图]， 一般情况下需要传入，若不传入baseMap则默认加载OSM（openStreetMap）地图
    projection 否 string "EPSG:4326" 地图使用的投影坐标系，可选"EPSG:4326"， "EPSG:3857"
    centerLng 否 number 0 地图初始化经度（遵循EPSG:4326数据格式）
    centerLat 否 number 0 地图初始化纬度（遵循EPSG:4326数据格式）
    zoom 否 number 0 地图初始化视图缩放值
    maxZoom 否 number 19.9 地图最大缩放层级
    minZoom 否 number 0 地图最小缩放层级
    extent 否 Array.<number> undefined 限制地图可见范围，数组[minx, miny, maxx, maxy]中四个值依次为X方向最小值，y方向最小值，x方向最大值，y方向最大值，通俗理解就是经纬度的最值，一般业务场景下请加上该配置项并设为[23, 2.022755, 162, 65.782116]限制在中国范围内
    controls 否 Array.<string> [] 默认无控件，数组可选字符 "Zoom"：缩放按钮控件， "Rotate"：旋转按钮控件， "Attribution"：右下角的地图信息控件， "FullScreen"：全屏控件， "MousePosition"：鼠标位置控件， "OverviewMap"：鸟瞰图控件， "ZoomSlider"：缩放滚动条控件， "ZoomToExtent"：缩放到设定区域控件 "ScaleLine"：比例尺控件， 详情查看 controlName
    isAllowDragRotate 否 boolean true 地图原生交互：是否允许拖拽旋转（shift+alt+drag）
    isAllowDragZoom 否 boolean true 地图原生交互：是否允许拖拽缩放（shift+drag）
    isAllowDragPan 否 boolean true 地图原生交互：是否允许拖拽移动
    isAllowDoubleClickZoom 否 boolean true 地图原生交互：是否允许双击缩放
    isAllowMouseWheelZoom 否 boolean true 地图原生交互：是否允许鼠标滚轮缩放

# 返回值

# Object

  • 类型：Object
  • 描述：地图初始化后返回一个对象，对象中包含下面两个字段
  • 详细属性：Object包含以下属性
    属性名 类型 描述
    TMap ol/Map 创建的地图实例
    baseLayer ol/Layer 创建的地图的底图实例
