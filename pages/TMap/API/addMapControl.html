# API / addMapControl

# 类型

(TMap:TMap,controlName:controlName,controlOption:controlOption )=>Control:Control

(地图实例化对象,控件名称,控件配置)=>控件实例化对象

# 描述

添加名称为controlName的控件到地图上，并且遵循配置controlOption

# 参数

# TMap

  • 类型：Object
  • 描述：地图实例对象（初始化地图返回值之一）
  • 是否必须：是
  • 详细属性：Map对象，详细请参照ol/Map

# controlName

  • 类型：string
  • 描述：控件名称
  • 是否必须：是
  • 详细属性：可选值如下
    控件名称 控件中文名称 简介
    "Zoom" 缩放按钮控件 在地图左上角添加两个按钮，分别为放大缩小功能
    "Rotate" 旋转按钮控件 在地图右上角添加一个按钮，在地图发生旋转后显示（视口上方偏离地图正北方），按钮箭头始终指向地理北方，点击后复位
    "Attribution" 地图信息控件 当地图图源为OSM时添加右下角地图角标信息，使用自定义图源时不显示
    "FullScreen" 全屏控件 地图右上角添加一个按钮，点击后会将地图扩展到全屏，再次点击回到原样
    "MousePosition" 鼠标位置控件 添加一个控件实时显示鼠标位置，默认显示在地图右上角，可配合配置项一起使用
    "OverviewMap" 鸟瞰图控件 地图左下角添加一个鸟瞰图控件，点击后显示或隐藏，拖拽鸟瞰图可实现地图联动移动
    "ZoomSlider" 缩放滚动条控件 地图左侧添加一个滚动条控件，拖动可实现视图放大缩小
    "ZoomToExtent" 缩放到设定区域控件 地图左上角Zoom控件下方位置添加一个按钮，上有字样E点击后缩放到指定范围
    "ScaleLine" 比例尺控件 地图左下角添加一个比例尺控件，显示单位为metric

# controlOption

  • 类型：Object
  • 描述：控件属性
  • 是否必须：否
  • 详细属性：Object包含以下属性
    属性名 是否必须 类型 默认值 描述
    mousePositionContainerId 否 string 地图所在目标容器id 添加鼠标位置控件时，可传入该参数指定鼠标位置显示容器,若不传则默认显示在容器右上角
    overviewBaseMap 否 string 添加鸟瞰图组件时，鸟瞰图底图配置名称 当前地图底图配置名称
    zoomToExtent 否 Array.[number] [-180, -90, 180, 90] 添加缩放到设定区域控件时，可传入该参数指定缩放范围，[minx, miny, maxx, maxy]，即经纬度最大最小范围

# 返回值

# Control

  • 类型：Object
  • 描述：地图控件实例化对象
  • 详细属性： Control对象，详情请查看ol/Control
