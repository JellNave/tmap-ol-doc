# 创建你的第一个地图

在上一章你已经学会了如何在项目中引入及使用TMap，现在你可以用TMap实现任何你想实现的功能

这里我们将从创建第一个地图开始，正式的使用TMap

# 容器

首先创建一个html文件，然后按上一章的内容进行资源的引入

准备一个具备宽高的div作为地图的容器

    <div id="mapContainer"></div>
1
  #mapContainer {
      width: 100vw;
      height: 100vh;
    }
1
2
3
4

# 配置

在script中定义一个常量配置对象,这个对象包含了如下属性

 const option = {
      targetContainerId: "mapContainer",//目标容器id
      centerLng: 104.076693,//中心点经度
      centerLat: 30.513315,//中心点纬度
      zoom: 18,//初始缩放层级
      baseMap: "dark",//底图配置名称
    };

# 调用

然后import引入initMap方法并调用，别忘了在script标签加type='module'

 <script type="module">
  import { initMap } from "../js/TMap.js";
   const option = {
      targetContainerId: "mapContainer",//目标容器id
      centerLng: 104.076693,//中心点经度
      centerLat: 30.513315,//中心点纬度
      zoom: 18,//初始缩放层级
      baseMap: "vector",,//底图配置名称
    };
    const map = initMap(option);
  </script>
